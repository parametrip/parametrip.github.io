Socializing kills is an interactive installation that criticizes the new social attitudes we have acquired as a result of the current pandemic. The aim of the exhibition is to make the visitors reflect on the social distancing measures and to make them rethink about their visions of the others.

This experience is carried out using a computer to read the data of the sensor and to generate the graphics, an ultrasonic sensor which measures distance between the user and the bust, a webcam to include the user’s silhouette into the projection, a projector to display the graphics and an Arduino microcontroller which sends the data from the sensor to the PC.
